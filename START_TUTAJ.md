# AI Council - Szybki Start

## 🚀 W 3 krokach:

### 1. Zainstaluj
```bash
pip install -r requirements.txt
```

### 2. Skonfiguruj
Edytuj `.env` - zamień `dummy-key` na prawdziwe API keys.

**Minimum**: 1 klucz (Gemini jest darmowy: https://aistudio.google.com/apikey)

### 3. Uruchom
```bash
python start.py
```

Otwórz: **http://localhost:8000**
